# 账号管理器管理后台 SPA

本目录包含账号管理器（Account Manager）管理后台的 React + TypeScript SPA 实现。管理后台用于查看系统状态、用户目录、用户设备、启用/禁用同步权限、撤销设备和查看审计日志。

## 实现者契约

- 仅通过 `@2fa/api-client` 与 `/v1/admin/*` 路径通信。
- 不得请求、存储或展示任何密文正文。`docs/openapi.yaml` 中的契约
  在管理路径上不返回这些内容；不要添加会返回密文的字段。
- 响应式布局支持 320px（移动端卡片流）到 1920px+（桌面端三栏），
  对应需求 8 和验收标准 AC13。
- 敏感操作（禁用用户、撤销设备、终止会话）必须通过确认模态框执行，
  且确认按钮使用高对比度的红色。

## 当前实现已提供内容

- `src/App.tsx`：管理后台主界面、登录、用户目录、设备抽屉、账号/关系元数据、只读关系图、确认弹窗、审计日志和系统状态。
- `src/main.tsx`：React 入口。
- `src/index.css`：Tailwind 样式入口和通用动画。
- `index.html`：中文页面标题与根节点。
- 已配置 `build` 和 `typecheck` 脚本。
